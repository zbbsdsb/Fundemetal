# Decision Cards Worksheet

A set of three decision cards for capital allocators to make structured go/no-go decisions. Each card targets a specific cognitive position.

## Complete Worked Example

**Scenario**: A premium AI-powered resume optimization service

### Card 1: Fatal Premise Validation

| Field | Value |
|---|---|
| Single fatal assumption | Job seekers are willing to pay $49/month for an AI tool that optimizes their resume for each application |
| Cheapest way to test | Run a landing page with a $49/month sign-up button and a $500 Google Ads campaign targeting "resume help" keywords. Measure click-to-signup conversion. |
| Cost of the test | $2,500 |
| Decision criterion | If conversion rate >= 2%, proceed to MVP build. If conversion rate < 1%, kill. |
| Time to answer | 14 days |

### Card 2: Resource Leverage

| Field | Value |
|---|---|
| Total capital available | $500,000 |
| Highest-leverage use of first $150,000 | Build the core AI parsing and optimization engine for the top 5 job industries (tech, healthcare, finance, marketing, operations) |
| Second-leverage use | Develop a self-serve onboarding flow and a sample "free tier" that optimizes one resume section to demonstrate value |
| Lowest-leverage use (do not spend here) | Custom enterprise integrations, native mobile apps, or multi-language support before PMF is validated |
| ROI estimate | For every $1 spent on the AI engine, we expect $8 in annual recurring revenue |

### Card 3: Minimum Sellable Unit

| Field | Value |
|---|---|
| Smallest sellable unit | Single-resume optimization with AI-powered keyword matching, formatting fix, and one cover letter draft. Delivered as a downloadable PDF + editable DOCX. |
| Price point | $29 (one-time) |
| Target customers | Mid-career professionals (3-15 years experience) actively applying to jobs on LinkedIn, Indeed, and Glassdoor |
| Sales channel | Landing page with Google Ads + organic LinkedIn content. Payment via Stripe checkout. |
| Success criterion | 200 sales in 30 days |
| Failure criterion | Fewer than 50 sales in 30 days |
