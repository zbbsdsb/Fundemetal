# γ Fundemetal — The Full Roster

> Drop-in README section. Extends the existing trilogy narrative to the whole library.

**Deep-level, methodological AI skill tools.** We ship only one kind of AI skill: cognitive protocols that force AI (and humans) into the right thinking discipline. Not task tools — *methods*. Not aphorisms — *protocols with falsifiable gates*.

Every skill in this repo obeys the same six-constraint DNA (cognitive protocol · anti-cheat hard gates · academic grounding · platform-agnostic Markdown · Swiss B&W · open source) and had to survive the same admission gate — [Curator](skills/curator/SKILL.md). What it turned away is logged in the [Graveyard](GRAVEYARD.md). What it lets through is below.

## The formations

Skills aren't a flat list; they're released in **formations** — squads that share a battlefield. Each formation attacks one layer of the thinking pipeline: `QUESTION → EVIDENCE → GENERATION → JUDGMENT → COLLECTIVE → META`. The full vulnerability catalog is the [Failure-Mode Registry](docs/FAILURE-MODES.md) — *OWASP for thinking*.

### ⓵ Build — the original trilogy *(shipped)*
Getting to a good plan.

| Skill | One line |
|---|---|
| **Decomposer** | Surface the unknown unknowns — turn "I don't know what I don't know" into actionable known unknowns. |
| **Tension Mining** | Find the force in the system — irreducible tensions → cross-domain invariants → mechanisms. |
| **Great Expectations** | Anti-consensus, anti-template planning with BANI + integrity gates. |

### ⓶ Falsify — the mirror trilogy
The plan you just built, now interrogated. Kill it, price it, autopsy it.

| Skill | One line |
|---|---|
| **[Pre-Mortem](skills/pre-mortem/)** | Assume the plan is already dead — write the obituary, ≥5 mechanisms, one self-inflicted. |
| **[Outside View](skills/outside-view/)** | Before you argue you're the exception, price the rule. Base rates before "we're different". |
| **[Resulting](skills/resulting/)** | Separate decision quality from outcome quality. A good result ≠ a good decision. |

### ⓷ Evidence — the anti-hallucination pair
Human self-deception and model hallucination are the same disease. Treat it.

| Skill | One line |
|---|---|
| **[Source Ledger](skills/source-ledger/)** | Tag every load-bearing claim: verified / inferred / reported / assumed. >40% assumed → no conclusion. |
| **[Disconfirmation Hunt](skills/disconfirmation-hunt/)** | Name what would prove you wrong, then go hunt it. Support isn't proof. |

### ⓸ Adversarial — the multi-agent cognition squad
Everyone's building agent *orchestration* (how to connect them). No one's building agent *cognitive hygiene* (how they don't infect each other with the same mistake). This is that flag.

| Skill | One line |
|---|---|
| **[Silent Start](skills/silent-start/)** | Everyone writes before anyone speaks. Kill the information cascade in ensembles and panels. |
| **[Red Cell](skills/red-cell/)** | Install a licensed adversary — and force the author to recite the hits they conceded. |
| **[Crux Finder](skills/crux-finder/)** | Locate the one belief whose flip changes the conclusion. Or prove it's a values divergence and stop. |

## Release order (don't ship them all at once)

Curation is the product. Suggested cadence:

1. **Sycophancy Breaker** solo, with its eval before/after image and the one-line hook ("a Markdown file that stops Claude agreeing by default"). It's the sharpest shareable unit — the hook that pulls people in.
2. **Falsify trilogy** — completes the Build↔Falsify symmetry; that mirror *is* the story.
3. **Evidence pair** — anti-hallucination is the market's live nerve.
4. **Adversarial squad** — plant the flag on multi-agent cognitive hygiene while it's empty ground.
5. **The rest** as single drops, each with its own before/after.

Every new skill must arrive with a 30-second before/after demo. No demo, no admission — that's Curator's first gate, and it doubles as the marketing asset.
