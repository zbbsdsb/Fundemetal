why they are one family

Shared DNA.

Not labels we stuck on — six design constraints every skill hard-obeys.

/01

Cognitive protocol, not a tool

Forces AI and humans into a specific thinking discipline, instead of executing an action for you.

/02

Anti-cheat hard gates

Red flags, banned words, acceptance metrics — stop the AI from taking shortcuts and shipping hollow output.

/03

Academic grounding

Kahneman, Kuhn, Cynefin, BANI — standing on frameworks that are already validated.

/04

Platform-agnostic

Markdown-first skill files — drop them into Claude, TRAE, Cursor, or Windsurf and they just work.

/05

Swiss B&W aesthetic

Minimal, restrained, high-contrast. Lighthouse 100, WCAG AA contrast.

/06

Open source (CC0)

Repository is CC0 1.0; skills keep their MIT license where noted. Fork it, cite it, remix it.

the architecture

Seven formations. One epistemology.

Every skill here is a cognitive protocol — the difference is which failure mode it attacks, and where in the thinking loop it lives.

BUILD
Decomposer → Tension Mining → Great Expectations. The starting sequence: surface unknowns, find the force, build the plan.
01 / 02 / 03
FALSIFY
Pre-Mortem · Outside View · Resulting. Before you commit, check: is the plan dead? What are the base rates? Did you confuse luck with skill?
06 / 07 / 08
EVIDENCE
Source Ledger · Disconfirmation Hunt. Every claim tagged and verified — then test the opposite before trusting your conclusion.
09 / 10
ADVERSARIAL
Crux Finder · Red Cell · Silent Start. Disagreement is an asset. Find the real crux, install a licensed adversary, kill the information cascade.
11 / 12 / 13
SINGLE
Question Autopsy · Quantity Quota · Trade-off Ledger · Second-Order · Steelman Forge · Stopping Rule. Six standalone operators — each attacks one specific cognitive trap.
14–19
AI-NATIVE
Sycophancy Breaker · Goal Anchor. Failure modes that only exist because the thinker is an LLM — agreement bias and goal drift.
20 / 21
META
Taboo · Curator · Insight Crystallizer · Shoulders of Giants. The layer that maintains the library itself — and curates its boundaries.

get started

Use a skill in under 30 seconds.

Copy the skill folder into your agent's skills directory. No build step — every protocol is plain Markdown.

        # Claude Code / Cursor / Windsurf — copy the skill into your agent

cp -r "skills/sycophancy-breaker/" ~/.claude/skills/
echo 'Follow the Sycophancy Breaker protocol in SKILL.md.'

# Any LLM (ChatGPT, Claude, Gemini) — paste SKILL.md into system prompt
# Every skill ships with an evals/ test suite.
cat skills/decomposer/SKILL.md
# Then paste it above your main instruction.
