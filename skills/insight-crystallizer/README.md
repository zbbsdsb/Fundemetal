# Insight Crystallizer

**A meta-protocol for turning tacit insights into reusable skills.** Recognize the moment a methodology is worth keeping, force the unnameable into words, crystallize it into a new skill.

*Honest. Reusable. Domain-agnostic.*

---

## What it is

Insight Crystallizer is not a tool for doing a task. It is a **meta-cognitive protocol** — a repeatable, teachable three-phase process for capturing the tacit "quality without a name" that emerges in the middle of real work, and fixing it as a skill you (or your agent) can reuse.

It pairs naturally with the rest of Fundemetal: where Decomposer, Tension Mining, and Great Expectations *use* methodologies, Insight Crystallizer *manufactures* them.

## Three phases

- **01 — Recognize**. Signal-driven, not request-driven. Five triggers (breakthrough after struggle, corrective redirection, recurring judgment, "I can't say why" moment, explicit request). Suppression rules keep it from nagging.
- **02 — Articulate**. The four-question structure — Context → Tension → Move → Boundary — forces the tacit into explicit language. A methodology with no stated failure mode is a slogan and is rejected.
- **03 — Crystallize**. Solidify into a standard skill, preserving the original source case (the anchor of tacit knowledge). Every reuse re-crystallizes.
